🖥️ Coordinate System in Pygame:

In Pygame, the coordinate system is a bit different from traditional Cartesian systems:

  • 📏 The origin (0, 0) is located at the top-left corner of the window.
  • 📐 X-axis: The x-coordinate increases as you move from left to right.
  • 📉 Y-axis: The y-coordinate increases as you move from top to bottom.

🔍 Although there are many new commands in this code that you haven’t learned yet, don't worry—you’ll learn them as we progress! For now, here’s a quick look at the key variables in the code:

  • 🐍 snake_x: The x-coordinate of the snake.
  • 🐍 snake_y: The y-coordinate of the snake.
  • 🖥️ SCREEN_WIDTH: Width of the game screen.
  • 🖥️ SCREEN_HEIGHT: Height of the game screen.
  • 🏃‍♂️ running: This variable holds the game status—if the game is running, it will be True; otherwise, False.
